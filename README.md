## 🚀 Quick Setup

Clone the repo and start exploring:

```bash
git clone https://github.com/Ruusheka/Estate.git
cd Estate
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser and dive in! 🌟
